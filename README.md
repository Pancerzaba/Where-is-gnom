## Zainnstaluj paczki
```
yarn install
```

## Uruchom projekt
Uruchom w konsoli 
```
yarn start
```
Zainstaluj i uruchom na telefonie aplikację Expo Go. 
Kliknij w "Scan QR Code" i zeskanuj kod QR
